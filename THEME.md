# SnapSurf 테마 가이드

SnapSurf는 **라이트 테마**와 **다크 테마** 두 가지 테마를 제공합니다.

## 🎨 테마 변경 방법

### 라이트 테마 (기본) 🌞
```bash
cp src/App-light.css src/App.css
```

### 다크 테마 🌙
```bash
cp src/App-dark.css src/App.css
```

변경 후 애플리케이션을 재시작하면 새로운 테마가 적용됩니다.

---

## 📋 테마 미리보기

### Light Theme (구글 스타일)
- **배경**: 화이트 (#ffffff)
- **주 색상**: 구글 블루 (#1a73e8)
- **텍스트**: 다크 그레이 (#202124)
- **보조 배경**: 라이트 그레이 (#f8f9fa)
- **테두리**: 실버 그레이 (#e8eaed)

**특징:**
- 깔끔하고 밝은 디자인
- 눈의 피로가 적음 (낮 시간 사용)
- 전문적이고 비즈니스적인 느낌

### Dark Theme (크롬 다크 스타일)
- **배경**: 다크 그레이 (#202124)
- **주 색상**: 블루 (#8ab4f8)
- **텍스트**: 라이트 그레이 (#e8eaed)
- **보조 배경**: 미드 그레이 (#292a2d)
- **테두리**: 다크 그레이 (#3c4043)

**특징:**
- 눈에 편한 다크 테마
- 야간 사용에 최적화
- 현대적이고 세련된 느낌

---

## 🎯 권장 사용 시나리오

### 라이트 테마 추천:
- 밝은 환경에서 작업할 때
- 낮 시간대
- 화면을 공유하거나 프레젠테이션할 때
- 텍스트 가독성이 중요할 때

### 다크 테마 추천:
- 어두운 환경에서 작업할 때
- 야간 또는 새벽 시간대
- 장시간 화면을 봐야 할 때
- 눈의 피로를 줄이고 싶을 때

## 🔧 개발자를 위한 정보

### 테마 파일 구조
```
src/
├── App.css          # 현재 적용된 테마
├── App-light.css    # 라이트 테마
└── App-dark.css     # 다크 테마
```

### 커스텀 테마 만들기
1. `App-light.css` 또는 `App-dark.css`를 복사
2. CSS 변수를 원하는 색상으로 수정
3. `src/App.css`로 복사하여 적용
